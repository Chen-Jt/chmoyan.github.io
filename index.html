Ubuntu英文服务器安装中文支持

  1. 安装中文支持： sudo apt-get install language-pack-zh-hant language-pack-zh-hans
  2. 添加中文支持：
    sudo vim /var/lib/locales/supported.d/local
    设置如下：
    en_US.UTF-8 UTF-8
    zh_CN.UTF-8 UTF-8
    zh_CN.GBK GBK
    zh_CN GB2312

运行sudo locale-gen 或者　sudo dpkg-reconfigure locales
3.　修改配置文件:
sudo vim /etc/environment
1、如果你需要一个纯中文的系统的话，设定LC_ALL= zh_CN.XXXX，或者LANG= zh_CN.XXXX都可以.
2、如果你只想要一个可以输入中文的环境，而保持菜单、标题，系统信息等等为英文界面，那么只需要设定 LC_CTYPE＝zh_CN.XXXX，LANG=en_US.XXXX就可以了。
我的配置如下：

PATH="/usr/local/sbin:/usr/local/bin:/usr/sbin:/usr/bin:/sbin:/bin:/usr/games:/usr/local/games"
LANG="en_US.UTF-8"
LANGUAGE="zh_CN:zh:en_US:en"
LC_CTYPE="zh_CN.UTF-8"

我们学校教务系统总共采取了三种措施来认证访问的合法性：
1,随机的Hash值
2,随机的Viewstate值
3,Referer值
第二条在我们参考的那篇文章中已经解决。

也就是当我们访问http://202.200.112.200 时它会跳转到一个“不确定”的页面，其实只要获取这个Hash值就可以了，按F12看看

发现Headers中多了个个Referer属性，这个值是记录你上次访问的页面地址，也就是你从那个页面跳转到这个页面的，要是不符合规则就会访问失败，因此在每次访问时要在Headers中更新Referer信息。

vim自动补全插件YouCompleteMe安装

YouCompleteMe简介

YCM基于语义引擎（比如C家族的libclang）提供了语义补全，是通过分析源文件，经过语法分析之后进行补全。对于C家族的语言这种基于语义的自动补全依赖于clang/llvm，其他语言 ,会调用vim设置的omnifunc来匹配，详细信息可以查看github

安装

使用Vundle来安装没安装Vundle的戳这里
安装好Vunle后在vimrc文件中添加Pluing 'Valloric/YouCompleteMe',执行PlugingInstall安装，要下载的文件较多，请耐心等待。
由于使用的是kali和ubuntu属于同系系统，可采用ubuntu系统的安装方式。
执行下面步骤前请确保

  1. 你的vim版本在7.3.598以上并且支持python2或python3
  2. 安装了cmake和python-dev,若没有安装，执行sudo apt-get install build-essential cmake,sudo apt-get install python-dev python3-dev。

希望支持c族语言则./install.py --clang-completer；
C# :./install.py --omnisharp-completer;
JavaScript: ./install.py --tern-completer;
如果安装过程中出错尝试使用root权限安装。

配置

打开默认的配置文件
vim ~/.vim/bundle/YouCompleteMe/third_party/ycmd/cpp/ycm/.ycm_extra_conf.py
在flags[ * ]数组的后面添加你想要的路径，例如: stdio.h等Ｃ语言的头文件包含在/usr/include中，那么您需要添加这样一条

注释掉下面这4行

接下来打开 ~/ 下的.vimrc文件添加let g:ycm_global_ycm_extra_conf = '~/.vim/bundle/YouCompleteMe/third_party/ycmd/cpp/ycm/.ycm_extra_conf.py'即指定默认配置文件路径。
